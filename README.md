##  Arquitetura das Pastas

A organização foi pensada para facilitar a navegação e o entendimento da hierarquia de pacotes do Java:

```text
JAVA_POSITIVO/                 # Pasta raiz do ambiente
└── java_positivo/             # Pasta do projeto principal
    ├── .gitignore             # Arquivos ignorados pelo Git
    ├── java_positivo.iml      # Arquivo de configuração (IntelliJ)
    └── src/                   # Pasta de código-fonte
        └── lista-01/          # Pasta da primeira lista de exercícios
            ├── ex01.java      #  ex01

      

        └── Main.java          # Classe principal de teste
```
## Como Rodar os Exercícios
Clone o repositório e abra no seu editor:
```text
git clone https://github.com/LuanNeuwirthC/AuladeJava.git
```

- Abra a pasta no VS Code
- Abra o arquivo .java desejado (ex: ex01.java) e clique no botão Run Java para rodar o código
